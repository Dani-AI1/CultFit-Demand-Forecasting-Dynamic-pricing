**Implement dynamic pricing strategies for fitness classes based on demand, time, and location**

In this project, you will help analyze attendance data of various fitness classes and develop a dynamic pricing model that will help Cult fit maximise attendance and increase their revenue per class.

**Objectives**

1. Identify high-demand class offerings across various locations and time zones.

2. Prepare the data for machine learning by handling data preprocessing, missing values, filtering outliers, etc.

3. Identify the correlation patterns for various demanded classes being offered by service provider.

4. Develop a pricing model to increase the price (from base price) for peak hours and High-in demand programmes for the Maximum number of bookees being booked and offer good discounts (from base price) for low in demand programmes to attract clients in order to generate overall revenue.

**Dataset**

The project is based on a 2 structured dataset (Class Apr-May 2018.csv and Class june 2018.csv) with features such as:

1. ActivitySiteID,	ActivityDescription,	BookingEndDateTime (Month / Day / Year),	BookingStartTime,	MaxBookees,	Number Booked	Price (INR)
